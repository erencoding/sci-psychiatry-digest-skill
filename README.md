# SCI 一区心理精神科文献速递 Skill

🌐 自动收集 JCR Q1 / 中科院 1 区心理学/精神病学高影响因子文献，验证所有链接后整理成飞书文档。

## 🚫 目标与禁止事项

**目标**：汇总有效的 SCI Q1 和中科院 1 区心理精神科文章

**禁止**：
1. 虚假信息（虚构文章标题、作者、期刊、IF、链接等）
2. 无效链接（404 失效且无法替换为 DOI 的条目必须删除）
3. 未经验证的条目

**核心原则**：宁缺毋滥——无法确认真实性时，宁可删除也不能写入虚假内容。

## 核心功能

- 支持 JCR Q1 和中科院 1 区双体系
- 所有链接输出前 100% 验证（200 OK 方可写入）
- 403 Embargo → 自动替换为 DOI
- 404 失效 → 自动删除并标注
- 输出带链接验证结果附录

## 使用方式

在 OpenClaw 中触发：
- "给我 SCI 一区心理精神科的文章"
- "中科院一区心理精神科文献"
- "心理学 Q1 期刊最新论文"

## 技术说明

- 链接验证：`curl -s -o /dev/null -w "%{http_code}" -L --max-time 10`
- 出版检索：PubMed/PMC + 出版社官网
- 输出格式：飞书云文档（Markdown）
